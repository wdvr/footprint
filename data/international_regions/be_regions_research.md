# Belgium Administrative Divisions - ISO 3166-2:BE

## Overview

Belgium is a federal state with a complex administrative structure reflecting its linguistic and cultural divisions. The country is divided into three regions and ten provinces, with the Brussels-Capital Region functioning at the provincial level without being a province.

**Total First-Level Divisions:** 11 (10 provinces + 1 capital region)

## ISO 3166-2:BE Code Structure

Each code consists of two parts separated by a hyphen:
- First part: `BE` (ISO 3166-1 alpha-2 code for Belgium)
- Second part: Three letters, where:
  - First letter indicates the region (`V` = Flemish, `W` = Walloon, `B` = Brussels)
  - Following letters derive from the subdivision name

## Regions (3)

| ISO Code | English Name | Official Name(s) |

| BE-VLG | Flemish Region | Vlaams Gewest |

Note: The Brussels-Capital Region (BE-BRU) operates without a province and assumes provincial responsibilities directly.

## Historical Notes

- In 1995, the former province of Brabant was split into three entities:
  - Brussels-Capital Region (BE-BRU)
  - Flemish Brabant (BE-VBR)
  - Walloon Brabant (BE-WBR)
- The original draft ISO standard used `BXL` for Brussels, but this was changed to `BRU` in the final standard
