# France Administrative Divisions - ISO 3166-2:FR

## Overview

France is divided into 18 administrative regions: 13 metropolitan regions (in Europe, including Corsica) and 5 overseas regions. The current regional structure results from the 2016 territorial reform which reduced the number of metropolitan regions from 22 to 13.

**Total Regions:** 18 (13 metropolitan + 5 overseas)

## ISO 3166-2:FR Code Structure

Each code consists of two parts separated by a hyphen:
- First part: `FR` (ISO 3166-1 alpha-2 code for France)
- Second part: Either:
  - Three letters for metropolitan regions (e.g., FR-ARA)
  - One digit + two letters for Corsica (FR-20R)
  - Numeric codes for departments
  - Two letters for overseas collectivities (matching their ISO 3166-1 alpha-2 codes)

### Notes on Corsica

Corsica (FR-20R) has a special status as a "territorial collectivity" (collectivite territoriale) since 1991, giving it more autonomy than mainland regions. The code "20R" uses the historic department prefix "20" that was divided into 2A and 2B in 1975.

## 5 Overseas Regions

These territories are both overseas departments (departements d'outre-mer) and regions. They also have their own ISO 3166-1 country codes.

| ISO Code | Name | ISO 3166-1 Code | Location |
|----------|------|-----------------|----------|
| FR-GF | Guyane (French Guiana) | GF | South America |
| FR-GP | Guadeloupe | GP | Caribbean |
| FR-MQ | Martinique | MQ | Caribbean |
| FR-RE | La Reunion | RE | Indian Ocean |
| FR-YT | Mayotte | YT | Indian Ocean |

Note: Mayotte became an overseas department in 2011.

## 2016 Regional Reform

The 2016 reform merged several regions:

| New Region | Former Regions Merged |
|------------|----------------------|
| Auvergne-Rhone-Alpes | Auvergne + Rhone-Alpes |
| Bourgogne-Franche-Comte | Bourgogne + Franche-Comte |
| Grand Est | Alsace + Champagne-Ardenne + Lorraine |
| Hauts-de-France | Nord-Pas-de-Calais + Picardie |
| Normandie | Basse-Normandie + Haute-Normandie |
| Nouvelle-Aquitaine | Aquitaine + Limousin + Poitou-Charentes |
| Occitanie | Languedoc-Roussillon + Midi-Pyrenees |

**Unchanged regions:** Bretagne, Centre-Val de Loire, Corse, Ile-de-France, Pays de la Loire, Provence-Alpes-Cote d'Azur
