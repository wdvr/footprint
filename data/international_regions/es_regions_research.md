# Spain Administrative Divisions - ISO 3166-2:ES

## Overview

Spain is divided into 17 autonomous communities (comunidades autonomas) and 2 autonomous cities in North Africa (Ceuta and Melilla). These are the primary subdivisions for ISO 3166-2:ES, along with 50 provinces.

**Total First-Level Divisions:** 19 (17 autonomous communities + 2 autonomous cities)

## ISO 3166-2:ES Code Structure

Each code consists of two parts separated by a hyphen:
- First part: `ES` (ISO 3166-1 alpha-2 code for Spain)
- Second part: One or two letters derived from the name

Note: Ceuta and Melilla are Spanish exclaves on the coast of Morocco. They have a special status as Autonomous Cities but are treated similarly to autonomous communities with a single province in some databases.

## Single-Province Autonomous Communities

Seven autonomous communities consist of only a single province. In these cases, the ISO code for the province is the same as the autonomous community:

| Community | Province Code |
|-----------|--------------|
| Asturias | ES-O |
| Balearic Islands | ES-PM |
| Cantabria | ES-S |
| La Rioja | ES-LO |
| Madrid | ES-M |
| Murcia | ES-MU |
| Navarra | ES-NA |

Note: The autonomous community codes (ES-AS, ES-IB, etc.) differ from the province codes due to ISO 3166-2 Newsletter II-1 (2010) changes.

## ISO Code Changes (2010)

In 2010, several autonomous community codes were changed to avoid conflicts with province codes:

| Autonomous Community | Old Code | New Code |
|---------------------|----------|----------|
| Asturias | O | AS |
| Cantabria | S | CB |
| La Rioja | LO | RI |
| Madrid | M | MD |
| Murcia | MU | MC |
| Navarra | NA | NC |
| Balearic Islands | PM | IB |

Province codes remained unchanged.

## Historical Notes

- The autonomous communities were established following the 1978 Spanish Constitution
- The Basque Country and Catalonia were the first to be formed in 1979
- The province system dates back to 1833
