# Geographic Data

This directory contains geographic data generated by scripts in `/scripts/`.

## Data Sources

### States/Provinces & Cities
**Source:** [dr5hn/countries-states-cities-database](https://github.com/dr5hn/countries-states-cities-database)
**License:** ODbL 1.0 (Open Database License)

- 5,296 states/provinces across 224 countries
- 153,728 cities across 218 countries

### UNESCO World Heritage Sites
**Source:** [Igor-Vladyka/realplanet](https://github.com/Igor-Vladyka/realplanet)
**License:** Public Domain (UNESCO data)

- 1,052 UNESCO sites across 192 countries

### National Parks
**Source:** [openshift-roadshow/nationalparks-js](https://github.com/openshift-roadshow/nationalparks-js)
**License:** Public Domain (GeoNames data)

- 2,893 national parks across 161 countries

## Regenerating Data

The data files are not committed to git (they're 40MB+). To regenerate:

```bash
# Fetch states, cities, and countries metadata
python scripts/fetch_geographic_data.py --all

# Fetch UNESCO sites and national parks
python scripts/fetch_landmarks.py --unesco --parks --ios
```

## iOS App Data

Optimized JSON files for the iOS app are stored in:
`ios/Footprint/Resources/GeoData/`

These are smaller, optimized versions suitable for bundling with the app:
- `world_states.json` - States/provinces (437 KB)
- `world_cities.json` - Top 100 cities per country (1.1 MB)
- `world_landmarks.json` - UNESCO sites + national parks (348 KB)
